# 📝 Task Manager Pro

A modern and responsive Task Manager application built using React.
This app allows users to efficiently manage daily tasks with features like adding, editing, deleting, and marking tasks as complete.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* 📌 Mark tasks as completed
* 🎯 Clean and responsive UI
* ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

* React (Functional Components + Hooks)
* Vite
* JavaScript (ES6+)
* CSS

---

## 📂 Project Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/task-manager-pro.git

# Navigate to the project folder
cd task-manager-pro

# Install dependencies
npm install

# Run the development server
npm run dev
```
