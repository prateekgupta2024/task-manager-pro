# 🚀 Task Manager Pro

A fully-featured and modern Task Manager application built with React, designed to help users efficiently manage their daily tasks with persistence, filtering, and a clean user experience.

---

## 🌐 Live Demo

👉 https://task-manager-pro-vert-two.vercel.app/

---

## ✨ Features

* ✅ Add, edit, and delete tasks
* 📌 Mark tasks as completed / pending
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Persistent data using LocalStorage
* ✏️ Inline editing for better UX
* 🎯 Clean and responsive UI
* ⚡ Fast performance with Vite

---

## 🧠 Key Concepts Used

* React Functional Components
* React Hooks (`useState`, `useEffect`)
* State management
* Conditional rendering
* Event handling
* LocalStorage integration
* Component-based architecture

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS

---

## 📂 Project Setup

```bash
# Clone the repository
git clone https://github.com/prateekgupta2024/task-manager-pro.git

# Navigate to the project folder
cd task-manager-pro

# Install dependencies
npm install

# Run the development server
npm run dev
```
